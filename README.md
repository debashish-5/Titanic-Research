## Project Overview

**[skillytixs-task-2](https://github.com/debashish-5/skillytixs-task-2)** is a data science project centered around exploring, analyzing, and visualizing the classic Titanic dataset. The primary goal of this repository is to clean passenger data, extract insights, and visually map out the correlations between passenger attributes and their survival rates.

---

## Key Features

* **Exploratory Data Analysis (EDA):** Leverages programmatic tools to uncover demographic trends, structural variations in ticket pricing, and underlying survival factors across the manifest.
* **Data Cleaning & Manipulation:** Features workflows for handling raw data inputs, resolving formatting discrepancies, and outputting updated structures.
* **Data Visualization:** Incorporates descriptive charts and plots to cleanly map out survival metrics against age, passenger class, and gender demographics.

---

## Repository Structure

The project relies entirely on Jupyter Notebooks for execution and consists of the following data and analysis files:

| File Name | Description & Purpose |
| --- | --- |
| **`Data Analysis.ipynb`** | Handles the initial loading, cleaning, and structural exploration of the passenger manifest. |
| **`Visualization.ipynb`** | Dedicated to plotting trends, distributions, and correlation graphs to make insights easily digestible. |
| **`titanic.csv`** | The raw foundational dataset containing the historic passenger manifest. |
| **`updates_titanic.csv`** | The cleaned or post-processed variant of the dataset generated from the analysis scripts. |

---

## Tech Stack
